# Prompt Diffing

Prompt diffing compares Intent IR across turns.

## Useful Diffs

- Verb changed: `explain` -> `create`
- Artifact evolved: `report` -> `slide deck`
- Audience changed: `CEO` -> `board`
- Constraint added: `brief` -> `brief + comprehensive`
- Gap resolved: `options` missing -> `options` filled
- Human posture changed: `uncertain` -> `expert`

## Use

Use diffing to understand whether the user changed tasks, refined the same task, or added constraints. Do not narrate the diff unless it helps the user.
